新文章格式的设定

Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量。

Hexo官方文档Front-matter

参数 描述 默认值
layout 布局
title 标题
date 建立日期 文件建立日期
updated 更新日期 文件更新日期
comments 开启文章的评论功能 true
tags 标签（不适用于分页）
categories 分类（不适用于分页）
permalink 覆盖文章网址

Hexo的布局

Hexo 有三种默认布局：post（博文）、page（页面） 和 draft（草稿），
它们分别对应不同的路径，page是直接储存在source下的。post和draft对应不同的子文件夹。
自定义的其他布局和 post 储存在同一文件夹。

Hexo生成文件名的设定

可以靠改变new_post_name来变更默认的文件名。
